## 📖 Usage

### Browse Examples
```bash
sherlock examples
```

### Copy an Example
```bash
sherlock examples --copy neural_network_demo
```

### Preview an Example
```bash
cd scenes
sherlock preview neural_network_demo.sherlock
```

### Render an Example
```bash
sherlock render neural_network_demo.sherlock -o my_video.mp4
```

## 💡 Tips

1. **Start Simple**: Begin with commented examples to understand syntax
2. **Experiment**: Modify copied examples to learn by doing
3. **Mix & Match**: Combine techniques from different examples
4. **Read Comments**: Many scenes include helpful inline explanations
5. **Check Duration**: Some examples are short (~5s), others longer (~30s)

## 🔍 Finding Examples

Use the search features in your editor to find examples by:
- **Concept**: Search `concept:` in scene files
- **Component**: Look for specific elements like `neural_network`, `graph`, `matrix`
- **Animation type**: Find `morph`, `follow_path`, `animate`, etc.

## 📝 Contributing Examples

Want to add your own example? Scene files should:
- Have a clear `concept:` description
- Be well-commented for learning purposes
- Demonstrate a specific feature or technique
- Run in under 30 seconds (for quick testing)
- Use meaningful element IDs
